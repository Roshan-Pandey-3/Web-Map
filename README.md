# 🗺️ Interactive Web Map for Amenities Around Paschimanchal Campus

An interactive, web-based map designed to help students, staff, and visitors easily locate essential amenities surrounding the Paschimanchal Campus (WRC) in Pokhara. 

### 🚀 [Click here to view the live Interactive Map](https://roshan-pandey-3.github.io/Web-Map/)

---

## 📍 Purpose & Overview

Moving into a new campus or navigating the surrounding area can be challenging. This project maps out key geographic points of interest around the Western Regional Campus (WRC) to make student life easier. It highlights essential categories such as:
* **Food & Dining:** Cafes, local eateries, and restaurants popular among students.
* **Daily Essentials:** Stationery shops, photocopying services, and grocery stores.
* **Health & Services:** Pharmacies, clinics, and ATM lounges.
* **Transportation:** Major bus stops and transit points near the campus.

## 🛠️ Features

* **Interactive Markers:** Clickable locations detailing the names and types of amenities.
* **Category Filtering:** Easily toggle between different types of places (e.g., food vs. shops).
* **Responsive Layout:** Optimized for mobile phones so students can use the map on the go.

## 🧰 Technologies Used

* **HTML5 & CSS3** - For the application structure, interface, and responsive map layout.
* **Web Mapping Library** - *(e.g., Leaflet.js / OpenStreetMap / Google Maps API - keep whichever you used!)*
